# Run Commands

## Training

### Train GAN (Default)
```bash
python galaxy.py
```

### Train Diffusion Model
```bash
python galaxy.py --diffusion
```

### Resume Training
```bash
python resume_training.py
```

## Utilities

### Check Data
```bash
python check_data.py
```

### Test Models
```bash
python test_models.py
```

### Generate from Checkpoint
```bash
python generate_from_checkpoint.py
```

### Debug Images
```bash
python debug_images.py
```

## Application

### Run Web App
```bash
python app.py
```
